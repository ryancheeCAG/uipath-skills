# SAP BAPI Activities Playbooks

**Investigation guide:** [investigation_guide.md](./investigation_guide.md) — data correlation rules and testing prerequisites for SAP BAPI Activities investigations

| Issue | Confidence | Description | Playbook |
|-------|:---:|-------------|----------|
| SAP — Connection / Logon Failure | Medium | A SAP BAPI activity faults while opening the RFC connection (`SAP Application Scope`), before any BAPI runs — usually a SAP .NET Connector exception via `SapConnectionService.Connect`: `SAP.Middleware.Connector.RfcConfigurationException` (config/params, e.g. `Not any kind of host is specified`), `RfcCommunicationException` (unreachable), `RfcLogonException` (bad credentials/authorization). Also UiPath-layer `Advanced Parameters has invalid parameter <param>...` (`SapActivityException`), `Connection could not be created.` / `Cannot create sap connection, connection info params are not set` / `Missing mandatory field: <field>` (`System.Exception`), or `System.TimeoutException`. The cause is the connection parameters / reachability / logon, never the BAPI. | [sap-connection-failed.md](./playbooks/sap-connection-failed.md) |
| SAP — BAPI Not Found or Not Named | Medium | The connection opened but the BAPI couldn't be resolved: `Function: <name> could not be created` (the function module doesn't exist in the connected system/client or the RFC user can't access it) or `BAPI name is null or empty` (no BAPI name configured). | [sap-bapi-not-found.md](./playbooks/sap-bapi-not-found.md) |
| SAP — Unsupported BAPI (Nested Complex Types) | High | The BAPI was found but its interface has nested complex structures/tables the activity can't marshal: `Unsupported BAPI. Contains nested complex types.` (`UiPath.SAP.BAPI.SapBapiExceptions.UnSupportedBapiException`). Deterministic for that BAPI — not a connection, auth, or input problem; needs a flatter BAPI or a different call mechanism. | [sap-unsupported-bapi.md](./playbooks/sap-unsupported-bapi.md) |
