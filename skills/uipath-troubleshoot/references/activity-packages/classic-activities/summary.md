# Classic Activities Playbooks

**Overview:** [overview.md](./overview.md) ‚Äî classic `UiPath.Core.Activities` (UI Automation + System/Core), failure families, and package boundaries

Covers the **classic** UI Automation (`UiPath.UIAutomation.Activities`) and System/Core
(`UiPath.System.Activities`) activities. For the modern "Next" UI activities see the
**ui-automation** package; for `Get Asset` / `Get Credential` (incl. `Get Robot Asset/Credential`)
see the **system-activities** package.

| Issue | Confidence | Description | Playbook |
|-------|:---:|-------------|----------|
| UI Element Not Found | Medium | A classic UI activity (`Click`, `Type Into`, `Send Hotkey`, `Take Screenshot`, `Wait UI Element Appear`, `Attach Browser/Window`, `Close Tab`) threw `SelectorNotFoundException` / "Cannot find the UI element corresponding to this selector" ‚Äî the target was not located within the timeout. Selector drift, application not open/ready, or wrong scope. | [ui-element-not-found.md](./playbooks/ui-element-not-found.md) |
| UI Activity Timeout | Medium | A classic UI activity threw `ActivityTimeoutException` ("Activity timeout exceeded") ‚Äî the element or state was never reached. Includes `Wait UI Element Appear` (element never appeared) and `Wait Image Vanish` (image still present after the timeout). | [ui-activity-timeout.md](./playbooks/ui-activity-timeout.md) |
| UI Element Interaction Failed | Medium | `ElementOperationException` from `Click` / `Type Into` / `Send Hotkey` / `Take Screenshot` ‚Äî the element was located but the action could not be performed (disabled, occluded, off-screen, or focus lost between find and act). | [ui-element-interaction-failed.md](./playbooks/ui-element-interaction-failed.md) |
| Browser Could Not Open or Attach | Medium | `BrowserOperationException` / invalid-browser error from `Open Browser`, `Close Tab`, or `Attach Browser` (Browser Scope) ‚Äî browser not installed/launchable, extension missing/disabled, browser crashed, or wrong browser type for the communication method. | [browser-open-or-attach-failed.md](./playbooks/browser-open-or-attach-failed.md) |
| Application Could Not Launch | Medium | `Open Application` / `Open Browser` failed to start the target. File/executable path not found, bad arguments, or "Both FileName and Arguments arguments are null" validation. The app never produced a window for the scope to attach to. | [application-launch-failed.md](./playbooks/application-launch-failed.md) |
| UI Activity Configuration Error | High | Design-time / validation error before execution: two mutually-exclusive options both set ("Only one of the {0} and {1} options can be set" ‚Äî e.g. SimulateClick + SendWindowMessages), conflicting `Selector` + `Browser`/`Window` on a scope, or a communication-method incompatibility on `Open Browser`. | [ui-activity-configuration-error.md](./playbooks/ui-activity-configuration-error.md) |
| Image Target Not Found / Did Not Vanish | Medium | `Wait Image Vanish` (or other classic image automation) did not behave as expected: the image was never matched, or it was still present after the timeout. Sensitive to display scaling, resolution, theme, and design-vs-runtime rendering. | [image-target-not-found.md](./playbooks/image-target-not-found.md) |
| File Operation Failed | Medium | `Rename File`, `Move File`, or `Append Line` faulted: "The source file does not exist.", "Destination should be a folder.", "File cannot be null", file already exists, path is a directory, access denied, or the file is locked by another process. | [file-operation-failed.md](./playbooks/file-operation-failed.md) |
| Invoke Workflow / Start Triggers Failed | Medium | `Invoke Workflow File` (or `Start Triggers`) faulted: workflow file not found, argument name/type/direction mismatch, isolated/elevated/session validation error, persistence not supported, or the invoked child workflow itself threw. | [invoke-workflow-failed.md](./playbooks/invoke-workflow-failed.md) |
| Invoke Code Failed | Medium | `Invoke Code` faulted: compilation failed ("No compiled code to run"), compilation errors in the snippet, unsupported language, or an exception thrown by the user's own code at run time. | [invoke-code-failed.md](./playbooks/invoke-code-failed.md) |
| Queue Operation Failed (Add/Get) | Medium | `Add Queue Item` or `Get Queue Item` / `GetQueueItem` faulted: a missing/null `QueueName`, queue not found (`Error code: 1002`), permission / HTTP / timeout error, or ‚Äî for Add only ‚Äî invalid or duplicate item-information keys. | [queue-operation-failed.md](./playbooks/queue-operation-failed.md) |
| Kill Process Failed | Low | `Kill Process` faulted with "Encountered errors while trying to kill a process" ‚Äî process not found, access denied (insufficient privileges or protected process), or errors across several matched processes. | [kill-process-failed.md](./playbooks/kill-process-failed.md) |
| For Each Row Failed | Low | `For Each Row` faulted: the DataTable is null/uninitialized, the iterator (`CurrentRow`) variable name is invalid, or an activity inside the loop body threw on a particular row. | [foreach-row-failed.md](./playbooks/foreach-row-failed.md) |
